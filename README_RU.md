# NewsHub - Современный новостной портал

## 🌟 Описание

NewsHub - это полнофункциональный новостной сайт с системой авторизации, комментариями, избранными новостями и административной панелью. Приложение построено на React с использованием Firebase Authentication и Firestore для хранения данных пользователей, комментариев и избранного.

## ✨ Основные возможности

### Для всех пользователей:
- 📰 Просмотр новостей из разных категорий
- 🔍 Фильтрация новостей по категориям (технологии, бизнес, спорт, развлечения, наука)
- 📱 Адаптивный дизайн для всех устройств

### Для авторизованных пользователей:
- 💬 Написание и удаление своих комментариев
- ❤️ Добавление новостей в избранное
- 👤 Личный профиль

### Для администраторов:
- ➕ Создание новых новостей
- ✏️ Редактирование существующих новостей
- 🗑️ Удаление новостей
- 🛡️ Модерация комментариев (удаление любых комментариев)

## 🔧 Технологический стек

- **Frontend:** React 18, TypeScript
- **Роутинг:** React Router 7
- **UI компоненты:** Radix UI, Material UI
- **Стилизация:** Tailwind CSS v4

- **База данных:** Firestore
- **Сборка:** Vite
- **Иконки:** Lucide React
- **Уведомления:** Sonner

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка Firebase

Следуйте подробной инструкции в файле [FIREBASE_SETUP.md](./FIREBASE_SETUP.md).

Кратко:
1. Создайте проект в [Firebase Console](https://console.firebase.google.com/)
2. Включите Authentication (Email/Password)
3. Создайте Firestore Database
4. Создайте файл `.env` с конфигурацией:

```env
VITE_FIREBASE_API_KEY=your_api_key_here
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 3. Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:5173

## 🔐 Система авторизации

### Firebase Authentication

Приложение использует Firebase Authentication для управления пользователями:

- **Регистрация:** Создание нового аккаунта с email и паролем
- **Вход:** Аутентификация существующих пользователей
- **Первый пользователь:** Автоматически получает роль администратора

### Данные пользователя в Firestore

```javascript
{
  uid: string,           // Уникальный идентификатор
  email: string,         // Email пользователя
  displayName: string,   // Отображаемое имя
  role: "admin" | "user", // Роль
  createdAt: string      // Дата создания
}
```

## 💬 Система комментариев

Комментарии хранятся в Firestore и связаны с:
- Конкретной новостью (newsId)
- Автором комментария (authorId)

**Возможности:**
- Авторизованные пользователи могут оставлять комментарии
- Пользователи могут удалять свои комментарии
- Администраторы могут удалять любые комментарии

## ❤️ Избранные новости

Функционал избранного позволяет пользователям:
- Добавлять новости в избранное одним кликом
- Просматривать все избранные новости на отдельной странице
- Удалять новости из избранного

Данные избранного хранятся в Firestore и связывают userId с newsId.

## 📝 Дополнительная информация

### Хранение новостей

По умолчанию новости хранятся в localStorage для упрощения демонстрации. Это позволяет:
- Работать без дополнительной настройки
- Быстро тестировать функционал
- При необходимости легко мигрировать в Firestore

### Интеграция с News API

Приложение поддерживает автоматическое обновление новостей из News API:
1. Получите API ключ на [newsapi.org](https://newsapi.org/)
2. Добавьте его в `.env` как `VITE_NEWS_API_KEY`
3. Новости будут автоматически обновляться каждый час

## 🐛 Решение проблем

### Firebase не инициализируется
- Проверьте правильность конфигурации в `.env`
- Убедитесь, что включены Authentication и Firestore в Firebase Console

### Ошибки при создании комментариев
- Проверьте правила безопасности Firestore
- Убедитесь, что пользователь авторизован

### Не работает избранное
- Проверьте правила безопасности для коллекции `favorites`
- Убедитесь, что пользователь авторизован
